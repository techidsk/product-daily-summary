About · 关于本站

关于 GitHub Trending 每日汇总

本站是一个独立维护的开源项目观察台。我们每天自动抓取 GitHub Trending 榜单， 把每日、每周、每月最受关注的开源项目整理成清爽、可读、可分享的形式， 并把每一天的榜单长期归档，方便你回看开源世界的演变。

GitHub Trending Daily is an independently maintained dashboard for the open-source world. Every day we collect the GitHub Trending list and organize the most-starred repositories into a clean, readable digest — daily, weekly and monthly — and archive each day so you can look back at how trends evolved over time.

我们做什么 · What we do

GitHub 官方的 Trending 页面只显示「当天」的榜单，过去的排名一旦滚动就再也找不回来。 本站解决三个问题：

  • 整理（Curate）：把原始榜单重新排版为按排名、星标增长、编程语言组织的易读列表。
  • 归档（Archive）：每天保存一份快照，并通过 Internet Archive 回填历史，让你能回看任意一天。
  • 分享（Share）：一键把当日 Top 5 导出为精美图片卡片，方便发到社交媒体或团队群。

数据从哪里来 · Where the data comes from

榜单数据来源于公开的 github.com/trending 页面， 历史数据通过 Internet Archive（Wayback Machine） 的公开存档回填。 本站不隶属于 GitHub，也未获 GitHub 或 Microsoft 的官方背书；所有仓库名称、描述与商标均归各自作者与权利人所有。 星标、Fork 等数值以抓取时刻为准，可能与 GitHub 实时数据存在延迟。

Ranking data is sourced from the public github.com/trending page, with historical days backfilled from the Internet Archive. This site is not affiliated with or endorsed by GitHub or Microsoft. All repository names, descriptions and logos belong to their respective owners.
