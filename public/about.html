About GitHub Trending Daily

GitHub Trending Daily is an independently maintained dashboard for the open-source world. Every day we collect the GitHub Trending list and organize the most-starred repositories into a clean, readable digest — daily, weekly and monthly — and archive each day so you can look back at how trends evolved over time.

本站是一个独立维护的开源项目观察台：每天自动抓取 GitHub Trending， 把每日 / 每周 / 每月最受关注的开源项目整理成清爽、可读、可分享的形式，并长期归档每一天的榜单。

What we do

GitHub's official Trending page only shows the current day — once the ranking scrolls, the past is gone. This site solves three problems:

  • Curate — reformat the raw list into a readable ranking organized by position, star growth and programming language.
  • Archive — save a snapshot every day and backfill history from the Internet Archive, so you can revisit any past day.
  • Share — export the daily Top 5 as a polished image card with one click, ready for social media or team chats.

Where the data comes from

Ranking data is sourced from the public github.com/trending page, with historical days backfilled from the Internet Archive (Wayback Machine). This site is not affiliated with or endorsed by GitHub or Microsoft. All repository names, descriptions and logos belong to their respective owners. Star and fork counts reflect the moment of capture and may lag GitHub's live data.

Languages

The interface is available in English, 简体中文 and 日本語. It follows your browser language automatically and can be switched manually at any time.
